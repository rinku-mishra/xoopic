Scalar BaseMCPackage::gasDensity
protected

When the Neutral Gas Density varies, the collision probability should depend on the maximum value of this density. We'll calcualte it at initialization. The maximum Neutral Gas Density "maxNGD" is equal to the "gasDensity" in the uniform case, and in the case of varying neutral gas density, the "gasDensity" will be initialized to the value of "maxNGD". dad 12/04/2000.
