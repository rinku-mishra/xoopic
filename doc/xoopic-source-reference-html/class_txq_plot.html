Detailed Description

The TxqPlot widget is the base class for plotting.

– This class creates a Qt Plot widget used for scientific plotting

Ryan McLean and Kelly G. Luetkemeyer

TxqPlot.h 907 1999-02-01 19:48:13Z kgl

The TxqPlot widget implements a scientific plot. It is capable of plotting both pixel and line style data.
