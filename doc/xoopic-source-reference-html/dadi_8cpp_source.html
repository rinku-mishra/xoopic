1 /**********************************************************************
2  This is a dynamic ADI solution of the equation
3  Del( epsilon * Del(phi) ) = rho
4  in z-r coordinates.
5  Neumann, Dirichlet, and symmetry boundary conditions are supported.
6  Symmetry boundary condition is only applied when r0=0, and the
7  neuman flag is specified there with zero slope on the electric field.
8 
9  dxdxu + dydyu = s
10 
11  The function is based on a Peaceman Rachford Douglas
12  advance of the parabolic equation:
13 
14  dtu = dxdxu + dydyu -s
15 
16  But with the time step dynamically chosen so as to speed convergence
17  to the dtu=0 steady state solution which is what we want. It is the
18  user's responsiblity to put the initial guess of the solution stored
19  in u when the function is called. u=0 or u=u(previous time step)
20  are possible choices.
21 
22  The function sends back the finishing iteration number, the finishing
23  normalized maximum error and location, and the failure code.
24 
25  The failure codes are:
26  ifail=0, success actually
27  ifail=1, had to discard too many times
28  ifail=2, used up the iterations
29 
30  Send in a negative tol_test if you want to freeze the time step to
31  the initial choice. Send in a 0 adel_t to let program pick an
32  initial del_t.
