1 /*
2  * VECTOR.H
3  *
4  * Class definitions for vectors, Vector3 and Vector2. The Vector3 class is
5  * a 3-component Vector3 representation with all the usual operators. Vector2
6  * is an implementation for 2-component vectors. Note that all operations
7  * are not valid on the latter.
8  *
9  * The Vector3 class is a 3-component vector representation
10  * encapsulating many of the standard mathematical vector operations.
11  * Here 'vector' refers to the mathematical concept of a vector,
12  * an ordered triplet representing a point in three-space.
13  *
14  * The Vector2 class is a 2-component vector representation
15  * encapsulating many of the standard mathematical vector operations.
16  * Here 'vector' refers to the mathematical concept of a vector,
17  * an ordered doublet representing a point in two-space.
18  *
19  * Revision/Programmer/Date
20  * 0.1 (JohnV, 12-07-92) Proto code.
21  * 0.2 (JohnV, 01-17-93) Fix case.
22  * 0.3 (JohnV, 02-20-93) Separate header file.
23  * 0.4 (JohnV, 02-27-93) Add Vector:: operator / (), -=() and +=().
24  * 0.5 (JohnV, 03-18-93) Add Vector2 class.
25  * 0.6 (JohnV, 08-02-93) Inline constructors, eliminate VECTOR.CPP, add
26  * Vector::operator *=(Vector&).
27  * 0.61 (JohnV, 11-22-93) Rename Vector->Vector3, add Vector3::norm(),
